## 播放与 Uni Playlist 验收

- QQ 歌曲数字 ID、MID、媒体 MID、`songType` 和文件规格分别保存，不能压成一个字符串。
- 统一流响应保留实际音质、VIP/试听边界、链接有效期、CDN 和必要请求头，可用于播放或下载。
- QQ 必须既可作为原始播放平台，也可作为网易云、Uni Playlist 等内容的回退目标；严格匹配仍使用标题、歌手、专辑、时长、ISRC 和版本标签。
- QQ 的公开歌单、账户创建歌单、收藏歌单和喜欢歌曲列表可作为 `platform=qq&type=...&id=...&account?` 的 Uni 导入来源；完整遍历上游分页并保留重复项与来源顺序。

- 每完成 3 个 QQ 上游公开方法，检查一次 `L-1124/QQMusicApi`；每 6–12 小时或用户主动要求时可提前检查。
